Default Function Arguments

Next >>

In this lesson, I learnt about using default function arguments to assign values to arguments that are undefine during declaration.

Code:

  1. let app = {};
  2.  
  3. /*
  4. * Here, we give the argument method a default value
  5. * of 'GET'. This value is only changed when the value for
  6. * method is passed. When not passed(meaning undefined) the value
  7. * remains unchanged . This means the argument method is optional.
  8. */
  9. app.checkRequestParams = (url, method = 'GET') => ({ url, method });
  10.  
  11. console.log(`Check request params result: ${JSON.stringify(app.checkRequestParams('google.com'))}`);
  12.  
  13. console.log(`Check request params result: ${JSON.stringify(app.checkRequestParams('google.com', 'POST'))}`);
  14.  
  15. // null will also be taken as a value for the argument
  16. console.log(`Check request params result: ${JSON.stringify(app.checkRequestParams('google.com', null))}`);
