Template Strings

Next >>

In this lesson, I learnt about template strings which are used in string interpolations for better single-line and multi-line string concatenation.

Code:

  1. let app = {};
  2.  
  3. app.getYearMessage = function () {
  4.     const year = new Date().getFullYear();
  5.  
  6.     /*
  7.     * Template strings are a syntactic sugar in ES6 where we concatenate
  8.     * strings by use of "${}" symbols like below. When define a template 
  9.     * string we also need "``" opening and closing symbols inorder for the
  10.     * concatenation to work. The below example can be inteprated as:
  11.     * return 'The year is "' + year + '" and 2 years later will be "' + (year+2) + '"';
  12.     */
  13.     return `The year is "${year}" and 2 years later will be "${year + 2}"`;
  14. }
  15.  
  16. console.log(app.getYearMessage());
  17.  
  18. console.log(`This is a message created in "${new Date().getFullYear()}"`);
