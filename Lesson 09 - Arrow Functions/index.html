In this lesson, I learnt about arrow functions and how they work.

Code:

  1. let app = {};
  2.  
  3. /*
  4. * Below implements an implicit return using a fat arrow function.
  5. * This is translated as:
  6. * const add = function(a, b) { return a + b; };
  7. */
  8. const add = (a, b) => a + b;
  9.  
  10. console.log(`Addition result: ${add(100, 33)}`);
  11.  
  12. /*
  13. * Here, we have an implicit return that returns an object like:
  14. * {sum: 13, multiplication: 30}
  15. * Below arrow function would be translated as:
  16. * const addMultiply = function (a, b) { return {sum: (a + b), multiplication: (a * b)}; };
  17. */
  18. const addMultiply = (a, b) => ({sum: (a + b), multiplication: (a * b)});
  19.  
  20. console.log(`Addition and multiplication result: ${JSON.stringify(addMultiply(10, 3))}`);
  21.  
  22. /*
  23. * With arrow functions we can also omit the parentheses whenever
  24. * our function has a single arguement.
  25. * Below would be translated as:
  26. * const double = function(number) { return 2 * number; };
  27. */
  28. const double = number => 2 * number;
  29.  
  30. console.log(`Double result: ${double(404)}`);
  31.  
  32. /*
  33. * In the case where our function has no arguements, we
  34. * must have parentheses added before the arrow.
  35. * Below would be translated as:
  36. * const tripple = function() { return 3 * 3 }
  37. */
  38. const tripple = () => 3 * 3;
  39.  
  40. console.log(`Tripple result: ${tripple()}`);
  41.  
  42. const numbers = [1, 2, 3, 4];
  43.  
  44. const double_numbers = numbers.map(number => double(number));
  45.  
  46. console.log(`Double numbers array result: ${JSON.stringify(double_numbers)}`);
  47.  
  48. /*
  49. * Advantage of using arrow functions is the use of lexical 'this' which
  50. * allows the value of 'this' to change in context to where we call it.
  51. * In ES5, that would not be possible and we would have to reassign 'this' to a variable outside
  52. * the function in context in order to access the values.
  53. */
  54. const team = {
  55.     members: ['Jane', 'Bill'],
  56.     team_name: 'Riff Raff',
  57.     teamSummary: function() {
  58.         return this.members.map((member) => {
  59.             return `${member} is on team ${this.team_name}`;
  60.         });
  61.     }
  62. }
  63.  
  64. console.log(`Team summary result: ${JSON.stringify(team.teamSummary())}`);
