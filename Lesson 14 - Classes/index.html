In this lesson, I learnt about classes and defining inheritance to child classes. Classes are mostly used in frameworks such as React, VueJS, Angular to extend functionality to components, directives e.t.c. We can use the rest/spread operator and destructuring in our classes as well.

Code:

  1. let app = {};
  2.  
  3. /*
  4. * In ES6 we define a class using the keyword 'class'. The below class has a
  5. * constructor that initializes the class object to be used in it's methods.
  6. */
  7. class Car {
  8.     /*
  9.     * We can use destructuring in the constructor to cherry pick the object properties
  10.     * and initialize them.
  11.     */
  12.     constructor({ brand, sound, color }) {
  13.         this.brand = brand;
  14.         this.sound = sound;
  15.         this.color = color;
  16.     }
  17.  
  18.     /*
  19.     * Using the keyword 'this' we can fetch the values of the object passed to the
  20.     * class and use them in our class methods(prototypes)
  21.     */
  22.     drive() {
  23.         return `The ${this.color} '${this.brand}' makes the sound '${this.sound}' when driving`;
  24.     }
  25. }
  26.  
  27. const car = new Car({ brand: 'Toyota', sound: 'vroom', color: 'red' });
  28. console.log(car.drive());
  29.  
  30. /*
  31. * In ES6 we define class inheritance using the keyword 'extends'. To initialize the inherited
  32. * properties and methods we call the 'super()' method in our child class constructor.
  33. */
  34. class Ford extends Car {
  35.     /*
  36.     * We can use destructuring and the rest operator in our constructor for initializing 
  37.     * the child class properties as well as the parent class properties.
  38.     */
  39.     constructor({ model, honk_sound, ...options }) {
  40.         super(options);
  41.         this.model = model;
  42.         this.honk_sound = honk_sound;
  43.     }
  44.  
  45.     honk() {
  46.         return `The ${this.color} '${this.brand} ${this.model}' makes the sound '${this.honk_sound}' when honking`;
  47.     }
  48. }
  49.  
  50. const ford = new Ford({ brand: 'Ford', model: 'Focus', sound: 'grrrr', color: 'blue', honk_sound: 'beep' });
  51. console.log(ford.drive());
  52. console.log(ford.honk());
