Rest|Spread Operator

Next >>

In this lesson, I learnt about the rest and spread operators. The rest operator is mostly used in context to methods where there is an unknown number of arguments passed. However the spread operator is used to add elements to an iterable collection such as arrays and even strings.

Code:

  1. let app = {};
  2.  
  3. /*
  4. * The rest operator allows us to define an infinite number of 
  5. * arguments to a function. We use the prefix '...' on an argument to
  6. * signify that there are an unknown number of arguments passed.
  7. */
  8. app.addNumbers = (...numbers) => {
  9.     // To access the value of numbers in the callback, we remove the prefix '...'
  10.     return numbers.reduce((sum, number) => {
  11.         return sum + number;
  12.     }, 0);
  13. };
  14.  
  15. let add_result = app.addNumbers(1, 2, 3, 4, 5, 6, 7);
  16.  
  17. console.log(`Add numbers result: ${add_result}`);
  18.  
  19. /*
  20. * The spread operator allows us to add elements to an
  21. * iterable collection e.g array, strings. The spread operator
  22. * is also used to flatten elements into a single array.
  23. * Here we use the spread operator to split the string 'foo'
  24. * which is flattened into a single array.
  25. */
  26. let str = 'foo';
  27. let chars = [ ...str ] // [ "f", "o", "o" ]
  28.  
  29. console.log(`Characters "${str}" result: ${JSON.stringify(chars)}`);
  30.  
  31. /*
  32. * Here we use the spread operator to flatten the two color arrays
  33. * into a single array result. This can be translated as:
  34. * primary_colors.concat(user_favourites);
  35. */
  36. const primary_colors = ['red', 'green', 'blue'];
  37. let user_favourites = ['orange', 'purple'];
  38.  
  39. let available_colors = [ 'white', 'black', ...primary_colors, ...user_favourites ];
  40.  
  41. console.log(`Available colors result: ${JSON.stringify(available_colors)}`);
  42.  
  43. /*
  44. * We can use array methods/prototypes to check and modify values
  45. * in the spread operator.
  46. */
  47. app.validateShoppingList = (fruit, vegetable, ...others) => {
  48.     const default_item = 'milk';
  49.  
  50.     // To access the value of others in the callback, we remove the prefix '...'
  51.     if(others.indexOf(default_item) < 0) {
  52.         return [fruit, vegetable, default_item, ...others];
  53.     }
  54.  
  55.     return [fruit, vegetable, ...others];
  56. };
  57.  
  58. let shopping_list = app.validateShoppingList('orange', 'kales', 'soap', 'toothpaste');
  59.  
  60. console.log(`Shopping list result: ${JSON.stringify(shopping_list)}`);
