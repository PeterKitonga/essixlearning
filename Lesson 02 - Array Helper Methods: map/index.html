Array Helper Methods: map

Next >>

In this lesson, I learnt about the map helper which helps modify the values in an array. This helper always returns a new array of the result and does not require Array.prototype.push()

Code:

  1. let app = {};
  2.  
  3. app.numbers = [2, 1, 4, 10, 5];
  4.  
  5. /*
  6. * The map helper generates a new array which becomes the result of
  7. * the block of code inside it. It is used to modify data in an array.
  8. * In this example we are doubling the numbers into a new array.
  9. */
  10. app.doubled_numbers = app.numbers.map(function (number, index) {
  11.     let doubled = number * 2;
  12.     return doubled;
  13. });
  14.  
  15. console.log('Result of doubling ' + JSON.stringify(app.numbers) + ' is ' + JSON.stringify(app.doubled_numbers));
  16.  
  17. app.cars = [
  18.     {model: 'Buick', price: 'CHEAP'},
  19.     {model: 'Velar', price: 'EXPENSIVE'},
  20.     {model: 'F-Pace', price: 'MODERATE'},
  21. ];
  22.  
  23. /*
  24. * The map helper can also be used to pluck specific values from an array
  25. * of objects. For example, below we only want the prices from the cars array.
  26. */
  27.  
  28. app.car_prices = app.cars.map(function (car) {
  29.     return car.price;
  30. });
  31.  
  32. console.log('Car prices are ' + JSON.stringify(app.car_prices));
